Features

This are the major key features of the kanban board tool

Add new task

This is the section where users can add their respective tasks to the board

Drag/Move a task

The user can move task from one board to another to track the progress of each task

Comments

The users can add comment to a task under each board for more explanation and communication

Delete Task

This is the section where users can delete their respective tasks on the board

About

One of the project tasks posted on the ALX tasks intranet served as the basis for this project. the area where we first learned about JavaScript and its uses. The idea of a web socket, which allows for duplex (bidirectional) communication without having to worry about response waiting time, was discovered about that time. On topics like JavaScript, React, Node, and other project-related topics, we all read articles and watched videos. We worked on this project for about three to four weeks. While creating these projects, we ran into many errors and issues, but we also learned a lot about web-sockets, CORS, npm, and other things. Without the chance provided to us by the ALX team for their support, this would not have been possible. Without the opportunity and privilege granted to us by the ALX team, which we are grateful for, this would not have been possible. Visit their website directly here.
